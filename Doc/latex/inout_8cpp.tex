\subsection{Detailed Description}
Manage input/output. 

We manage here the following operations\+:
\begin{DoxyItemize}
\item Read nominal instance from disk (both OR Library and Avella). See the introduction part of \doxyref{rcflp.\+cpp}{p.}{rcflp_8cpp} to see how the costs $c_{ij}$ are managed in the two instance types.
\item Read parameters for the different support sets. Currently, we read parameters for the following sets\+:
\begin{DoxyItemize}
\item Ellipsoidal support set. See \doxyref{read\+\_\+parameters\+\_\+ellipsoidal()}{p.}{inout_8cpp_ad0cfc2a0a7960353ae10a37028cf97aa}
\item Box support set. See \doxyref{read\+\_\+parameters\+\_\+box()}{p.}{inout_8cpp_a62d78cce6f4f2da0b01932777977f854}
\item Budget support set. See \doxyref{read\+\_\+parameters\+\_\+budget()}{p.}{inout_8cpp_a2f56c555090ec900ef22670c4fc774e0} 
\end{DoxyItemize}
\end{DoxyItemize}

\item forth row \+: $L \geq 1$, i.\+e., number of budget constraints.
