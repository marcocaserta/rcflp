\subsection{Detailed Description}
Manage input/output. 

We manage here the following operations\+:
\begin{DoxyItemize}
\item Read nominal instance from disk (both OR Library and Avella). See the introduction part of \doxyref{rcflp.\+cpp}{p.}{rcflp_8cpp} to see how the costs $c_{ij}$ are managed in the two instance types.
\item Read parameters for the different support sets. Currently, we read parameters for the following sets\+:
\begin{DoxyItemize}
\item Ellipsoidal support set. See \doxyref{read\+\_\+parameters\+\_\+ellipsoidal()}{p.}{inout_8cpp_a4533d4b560d6c318d2e26bea2e0317c2}
\item Box support set. See \doxyref{read\+\_\+parameters\+\_\+box()}{p.}{inout_8cpp_a6e694ece3832c26a48b6a156e1c925b5}
\item Budget support set. See \doxyref{read\+\_\+parameters\+\_\+budget()}{p.}{inout_8cpp_ac6dd46d8d9977c73586196164cab23f3} 
\end{DoxyItemize}
\end{DoxyItemize}

Read parameters to define the Box support. 

The file \textquotesingle{}params\+Box.\+txt\textquotesingle{} has the following format\+:
\begin{DoxyItemize}
\item first row \+: {\ttfamily \+\_\+epsilon}, i.\+e., the parameter used to define the with of the box. Given a nominal demand value $d_j$, we define the interval around $d_j$ as\+: \[ (1-\_epsilon)*d_j <= d_j <= (1+\_epsilon)d_j, \quad 0 \leq \epsilon \leq 1 \]
\end{DoxyItemize}

M\+A\+R\+CO 11.\+02.\+20\+: This is no longer used, since the only parameter of a box set, i.\+e., epsilon, is now read from command line with flag -\/e.

{\footnotesize\ttfamily void read\+\_\+parameters\+\_\+budget (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Read parameters to define the budget support. 


\begin{DoxyItemize}
\item File \+: {\ttfamily params\+Budget.\+txt}
\begin{DoxyItemize}
\item first row \+: {\ttfamily \+\_\+epsilon}, as above
\item second row\+: {\ttfamily \+\_\+delta}, i.\+e., how the $b_l$ value (the r.\+h.\+s. value of each budget constraint) is defined. We define $b_l$ as a percentage of the total demand of customers included in the budget constraint, i.\+e.\+: \[ b_l = \delta* (\sum_{j \in B_l} d_j), \quad 0 \leq \delta \leq 1 \]
\item third row \+: {\ttfamily \+\_\+gamma}, i.\+e., percentage of columns included in each budget constraint\+: \[|B_l| = \lfloor \_gamma n \rfloor, \quad 0 \leq \gamma \leq 1\]
\item forth row \+: $L \geq 1$, i.\+e., number of budget constraints. 
\end{DoxyItemize}
\end{DoxyItemize}
