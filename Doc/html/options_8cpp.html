Detailed Description

Read options from command line.

Command line options are:

  • -h : help (visualize the list of options)
  • -i : inputfile
  • -t : instance type:
    1. orLibrary
    2. Avella
  • -v : version:
    1. Single-source
    2. Multi-source
    3. Ellipsoidal
    4. Polyhedral
  • -u : uncertainty set
    1. Box uncertainty set
    2. Budget uncertanty set
  • -r : read from disk
    1. 0 No: A new Budget set is generated and stored
    2. 1 Yes: The Budget set is read from disk
  • -e : epsilon, i.e., the size of the uncertainty set used during the optimization phase. Typically, .

Note: The meaning of the flag -r is as follows. When a budget uncertainty set is used, we randomly select which customers, i.e., columns, will be included in the budget constraint. Next, this set, , is used to find a robust optimal solution for a given . If we want to compare solutions obtained for different values of , we need to ensure that the same budget constraint(s) is(are) used in the optimziation phase. Otherwise, the comparison would be meaningless.
