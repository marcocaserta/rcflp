Detailed Description

Read options from command line.

Command line options are:

  • -h : help (visualize the list of options)
  • -i : inputfile
  • -t : instance type:
    1. orLibrary
    2. Avella
  • -v : version:
    1. Single-source
    2. Multi-source
    3. Ellipsoidal
    4. Polyhedral
  • -u : uncertainty set
    1. Box uncertainty set
    2. Budget uncertanty set
