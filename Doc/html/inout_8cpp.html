Detailed Description

Manage input/output.

We manage here the following operations:

  • Read nominal instance from disk (both OR Library and Avella). See the introduction part of rcflp.cpp to see how the costs are managed in the two instance types.
  • Read parameters for the different support sets. Currently, we read parameters for the following sets:
    • Ellipsoidal support set. See read_parameters_ellipsoidal()
    • Box support set. See read_parameters_box()
    • Budget support set. See read_parameters_budget()
